Consulta de Bitácora Autorizaciones

Esta función permite consultar en detalle las autorizaciones que han sido eliminadas por el sistema, porque han cumplido su plazo para ser cruzadas o porque ya fueron procesadas; dependiendo de si se está utilizando como criterio o es el resultado de la misma, estos los campos pueden permitir ingresar los datos correspondientes o ser simplemente de salida.

Filtro: El formulario cuenta con un filtro inicial de búsqueda, necesario para la consulta de la información:

Número tarjeta

En este campo se captura el número de tarjeta del cliente a quien se desea consultar, una vez ingresado se despliega el nombre del mismo.

Establecimiento

Si la entidad desea realizar la consulta de las autorizaciones por código de establecimiento, en este campo se captura su código de identificación, desplegándose el nombre del mismo.

Número de autorización

En este campo se captura el número de autorización sobre el cual se desea realizar la consulta.

Estado

Campo en el que se captura el estado de la autorización sobre el cual se desea realizar la consulta, en donde el código "A" significa las autorizaciones que se encuentran activas es decir, no han sido cruzadas o aplicadas y "P" que indica que las autorizaciones han sido validadas antes de ser tenidas en cuenta en el proceso de Actualización de saldos.

Código respuesta

En este campo se captura el código de respuesta correspondiente a la autorización sobre la cual se desea realizar la consulta.

Código del proceso

Este campo hace referencia al código con el cual la entidad procesadora identifica la transacción autorizada.

Fecha hora transacción

El campo Fecha hora transacción acepta 19 caracteres en formato AAAA-MM-DD HH:MM:SS

Fecha de proceso

Campo que captura la fecha en la cual la entidad cargó dicha autorización.
